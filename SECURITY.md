# Seguridad y datos confidenciales

Este repositorio es una plantilla de referencia. No incluye:

- Archivos JSON de cuentas de servicio de Google Cloud
- Contraseñas ni tokens
- Datos de transacciones reales

Configura localmente las variables de entorno descritas en `Client/SETUP.md`. No subas credenciales al repositorio.

Si alguna vez se expuso un nombre de clave o ruta en el historial de Git, rota la clave en la consola de GCP.
